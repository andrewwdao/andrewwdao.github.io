Our project focused on developing a software tool for generating 3D network coverage maps and executing network optimisation to find a cost-optimal path from the backhaul networks to the demand locations.
We have won the **Telstra Excellence in Learning and Application** in the 2023 Graduate Capstone Project Showcase.
